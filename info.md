# Ya2DLNA Streaming

Интеграция для Home Assistant, позволяющая перенаправлять аудио с Яндекс Станции на DLNA-устройство.

## Возможности

- Выбор Яндекс Станции (источник звука) из сущностей media_player.
- Выбор DLNA-устройства (приёмник) из сущностей media_player.
- Управление стримингом через переключатель (switch).
- Опциональное отключение динамика Яндекс Станции во время стриминга.
- Поддержка автоматизаций.

## Установка

### Через HACS (рекомендуется)

1. Добавьте этот репозиторий как пользовательский репозиторий в HACS.
2. Найдите "Ya2DLNA Streaming" в разделе "Интеграции".
3. Нажмите "Установить".
4. Перезагрузите Home Assistant.
5. Добавьте интеграцию через UI (Настройки → Устройства и службы → Добавить интеграцию → Ya2DLNA Streaming).

**Примечание:** Если возникает ошибка "Server disconnected", убедитесь, что в репозитории создан релиз с тегом (например, `v1.0.0`). HACS требует наличия релиза для загрузки интеграции.

### Вручную

Скопируйте папку `custom_components/ya2dlna` в директорию `custom_components` вашего Home Assistant.

## Настройка

После добавления интеграции вам будет предложено выбрать сущности Яндекс Станции и DLNA-устройства. После сохранения появится switch для управления стримингом.

## Требования

- Работающий сервер ya2dlna_streaming (можно запустить через аддон или вручную).
- Токен Яндекс Музыки.
